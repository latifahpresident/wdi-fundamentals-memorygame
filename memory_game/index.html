Instructions

Concentration, also known as Match Match, Memory, Pelmanism, Shinkeisuijaku, Pexeso or simply Pairs, is a game in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. The objective of the game is to turn over pairs of matching cards.

Description

The Memory Game is fun for people of all ages and can also help build up memory and recall functions in the brain. The game is played by flipping up two random card. If the cards are a match the player gets to keep them. If the cards aren't a match the players flips it back face down, then it's the next players turn.
